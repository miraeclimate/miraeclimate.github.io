Mirae-WAVE

기상기후 수치모델 자료, 천리안 위성 자료, GTS관측 자료, 태풍 분석 정보를 열어 지구본 또는 Box 형태의 지도 공간에 3차원으로 데이터를 표출하는 기상기후자료 시각화·분석 환경

개요

Mirae-WAVE는 기상수치모델자료와 같은 대용량 데이터 세트를 3차원으로 시각화하는 Windows 기반 시스템으로 기존 2D 기반의 분석 차트에 비해 직관적이면서도 실제 대기구조를 표현하는데 뛰어납니다. 대화형 인터페이스를 통해 지구본 또는 Box 공간 위에 기상기후자료 시각화에 필요한 다양한 차트를 중첩하여 표출하며, 분석 이미지를 저장하거나, 회전 · 시간 · 고도에 따른 변화를 동영상으로 저장할 수 있습니다.
기상청(KMA) 선진예보시스템에서 많은 연구자들과 예보관들이 수치모델자료 분석과 예보 분석을 위해 Mirae-WAVE의 또다른 버전인 Gloview를 사용하고 있습니다.

버전 정보

  1. Grib file, nc file 형식 지원
  2. 변수 그루핑, 필터링 기능
  3. 평면도·연직단면도·흐름도·연직흐름도·3D Iso-surface·3D Streamline 차트 및 중첩 기능
  4. 시간 탐색, 고도 탐색, 자동 재생 기능
  5. 좌표계 변환( Globe <-> Box ), 육지·해상 마스킹, 조명·시점·원근변환 기능
  6. 레이어 속성·색상 테이블 사용자정의, 설정 저장하기·불러오기 기능
  7. 이미지·동영상 저장 기능
